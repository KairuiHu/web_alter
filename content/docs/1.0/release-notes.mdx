## üî• Synvo API v0.1.0

**Release Date:** 11:00 PM, October 26, 2025

### üöÄ Initial Features

This is the first release of Synvo API, which lays the foundation for building and managing agentic context within AI applications. Key features include:

- **Parallel Processing**: The backend now supports up to 10 concurrent users performing file uploads and queries simultaneously.
- **Factual Contextualization**: The algorithm supports basic factual contextualization, enabling users to search for relevant information from uploaded files based on their queries.
- **Console Dashboard**: The frontend integrates key functionalities such as:
  - **Upload Context**: Allows users to upload files for context creation.
  - **Get Context**: Enables users to retrieve relevant context based on uploaded documents.
  - **View Context**: Provides a view of contextualized data related to specific queries.
- **Agent-friendly JSON output**: The context is provided in an agent-friendly JSON format, tailored for efficient integration with AI agents.

### üß∞ Known Limitations

- **Contextualization Scope**: The current version focuses primarily on basic factual contextualization, especially on localization of key information.
- **Limited UI Features**: As the first release, the Console Dashboard is minimal and will evolve with future updates to offer more advanced features.
